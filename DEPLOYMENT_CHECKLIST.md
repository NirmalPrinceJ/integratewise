# Deployment Checklist

## Pre-Upload Checklist

- [ ] All HTML files are ready (6 pages)
- [ ] styles.css is complete
- [ ] script.js is complete
- [ ] images/logo folder contains all SVG logos
- [ ] .htaccess file is ready
- [ ] Contact information updated (if needed)

## Upload Steps

1. [ ] Log into cPanel
2. [ ] Open File Manager
3. [ ] Navigate to public_html
4. [ ] Upload all HTML files
5. [ ] Upload CSS and JS files
6. [ ] Upload .htaccess file
7. [ ] Create images/logo folder structure
8. [ ] Upload logo SVG files
9. [ ] Set file permissions (644 for files, 755 for folders)

## Post-Upload Testing

- [ ] Homepage loads correctly
- [ ] All navigation links work
- [ ] Logo displays in header
- [ ] Logo displays in footer
- [ ] Favicon appears in browser tab
- [ ] CSS styles apply correctly
- [ ] JavaScript functions work
- [ ] Mobile responsive design works
- [ ] Forms display correctly
- [ ] All images load

## Configuration

- [ ] Domain points to correct folder
- [ ] SSL certificate installed
- [ ] HTTPS redirect enabled (uncomment in .htaccess)
- [ ] Email accounts created
- [ ] Contact forms configured
- [ ] Analytics tracking added

## Final Steps

- [ ] Test on multiple browsers
- [ ] Test on mobile devices
- [ ] Submit to Google Search Console

- [ ] Share website URL
- [ ] Update social media profiles

## Support

If issues occur:
1. Check cPanel error logs
2. Verify file permissions
3. Test in incognito mode
4. Check browser console for errors
5. Contact hosting support if needed
