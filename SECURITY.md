# Security Controls

- **Rotation Policy:** Secrets rotated every 90 days or on incident; enforced by VaultGuard with 100% compliance tracking.
- **RBAC:** Roles: admin (all), operator (runs/workflows), analyst (read health/events), auditor (read-only logs). Least-privilege enforced per integration scope.
- **Audit Retention:** Run/access logs 400 days; webhook logs 30 days; export available on request.
- **Compliance Roadmap:** SOC2 Type II underway; GDPR DPA templates provided; regional workers for residency; HIPAA not in scope.
