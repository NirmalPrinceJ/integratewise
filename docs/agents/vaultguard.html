Agent Spec

VaultGuard

Secrets rotation and access attestations.

Inputs

secrets_vault entries, rotation_policy_days, rbac assignments, audit_retention config.

Outputs

rotation events with checksum, rotated_at, expires_at; access attestation logs with actor, scope, duration.

Webhooks

rotation.required, rotation.completed, attestation.requested events emitted with vault_id and policy metadata.

SLAs

Rotation compliance 100%; uptime >99.9%; rotation attempts capped at 3 with exponential backoff; audit logs retained 400 days.

Versioning

Policy schema versioned; major when rotation cadence fields change; minor for additional audit attributes.
