Agent Spec

SuccessPilot

Predictive health scoring built for mid-market CS teams.

Inputs

accounts, contacts, product_usage (jsonb), support_cases, nps_responses, contract_terms.

Outputs

health_scores rows with score (0-100), drivers (jsonb), model_version, computed_at, ttl_minutes.

Webhooks

health_score.updated (payload: account_id, score, drivers, model_version, expires_at); dispatched per workspace subscription.

SLAs

Compute <120s; accuracy ≥85% against labeled training sets; uptime >99.9%; refresh cadence default 6 hours; lead time ≥30 days for model changes.

Versioning

Semantic model_version (major.minor.patch). Major introduces driver schema changes; minor adjusts weights; patch fixes defects. Backfill executed on major/minor.
