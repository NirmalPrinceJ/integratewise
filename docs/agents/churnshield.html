Agent Spec

ChurnShield

Risk detection and proactive tasking.

Inputs

accounts, contacts, product_usage deltas, support backlog, sentiment labels, contract milestones.

Outputs

risk_signals (jsonb) attached to accounts; tasks pushed to target_workspace with priority and due_at.

Webhooks

risk.created and risk.resolved payloads include account_id, severity (low/med/high), reason_codes, recommended_actions.

SLAs

Compute <120s; false-positive rate ≤15%; uptime >99.9%; response fan-out per workspace capped at 50 tasks/min.

Versioning

Model version pinned per workspace; upgrades announced ≥30 days prior. Backtests stored with checksum for audit.
