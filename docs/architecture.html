Architecture

Atlas Spine Deployment

Edge workers fronted by Cloudflare; data plane in regional KV + R2 buckets; Postgres for configuration and audits; all traffic TLS 1.2+.

Health Compute Window

SuccessPilot runs on rolling 15-minute windows with batch backfill every 6 hours; max compute per account <120 seconds.

Retry/Backoff

Exponential backoff (2^n seconds up to 5 retries) for connectors; idempotent run_ids prevent double writes; dead-letter queue alerts routed to Ops.

Observability

Structured logs per run, metrics exported to Prometheus-compatible endpoint, alert policies tied to managed ops tiers.
