Security

Rotation Policy

All credentials in secrets_vault rotated every 90 days or immediately on incident; VaultGuard enforces rotation and reports compliance 100%.

RBAC

Roles: admin (all resources), operator (runs/workflows), analyst (read health_scores/events), auditor (read-only logs). Least-privilege enforced per integration scope.

Audit Retention

Run and access logs retained 400 days; webhook deliveries stored 30 days; exports available on request.

Compliance Roadmap

SOC2 Type II in-progress; HIPAA not in scope; GDPR DPA templates provided; data residency options via regional workers.
