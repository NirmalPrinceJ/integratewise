Rotation policy

secrets_vault entries rotate every 30 days by default; critical adapters every 14 days. Failed rotations retried 3x with exponential backoff.

RBAC roles

viewer (read-only), operator (run workflows), admin (configure adapters and secrets), auditor (export logs only). Scoped per workspace.

Audit retention

Rotation and access logs retained 400 days; workflow and run logs retained 180 days; exportable CSV + signed checksums.

Compliance roadmap

SOC 2 Type II evidence collection in progress; ISO 27001 mapping drafted; DPAs and subprocessor list available on request.
